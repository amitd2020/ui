Introduction

DATAGARDENER APIwill provide you all the data needed for the analysis of a Company, Director or PSC.

Why use this?

  • You will get all the company data on the go.
  • You will get the JSON file of the company data at your own system.
  • No need to search the company manually, just use the API service with the company number and data is all yours.

What is included?

All the data of a company is included in this service, Such as:
  • Company Details
  • Financial Data
  • Document Data
  • Related Companies and Directors Data
  • Corporate Land Data and many more.

Steps to implement it?

  • Be ready with your User ID, API Access Token and API link provided by us.
  • Select any company you want and extract it's Company Number.
  • Now you are ready to use this service, just input all the three parameters (companyNumber , accessToken , userId).

Company Detail

				GET/dg-api/getCompanyByCmpNo/'company number'

Sample Response

				
					
						{{companyDetail | json}}

Company Key Financials

				GET/dg-api/getFinancialData/'company number'

Sample Response

				
					
						{{financialData | json}}

Company Documents

				GET/dg-api/getListOfCompanyDocuments/'company number'

Sample Response

				
					 
						{{companyDocument | json}}

Related Company and Directors

				GET/dg-api/getRelatedCompaniesAndDirectorsByCmpNo/'company number'

Sample Response

				
					
						{{relatedCompaniesAndDirector | json}}

Complete Company Data

				GET/dg-api/getcompleteInformation?token='token_number'&number='company_number'

Sample Response

				
					
						{{companyCompleteInfo | json}}
